# Maximum.md парсер категорий товаров

Универсальный Python-скрипт для парсинга **любой категории товаров** с сайта **maximum.md**.
Собирает данные со всех страниц категории, скачивает изображения и сохраняет всё в Excel.

Категория задаётся **одной ссылкой** в коде.

---

## Что делает

- Открывает указанную категорию товаров
- Проходит по **всем страницам пагинации**
- Парсит только товары **в наличии**
- Сохраняет для каждого товара:
  - изображение
  - название
  - текущую цену
  - старую цену (если есть)
  - ссылку на товар
- Скачивает изображения в папку `images/`
- Формирует Excel-файл с картинками внутри ячеек

---

## Настройка категории

В файле `maxparser.py` меняется только эта строка:

```python
BASE_URL = "https://maximum.md/ru/kompyuternaya-tehnika/monitory/monitory/"
```

Можно подставить **любую категорию** с пагинацией:
- ноутбуки
- видеокарты
- бытовая техника
- аксессуары

## Установка

### 1. Скопировать проект
```bash
git clone <repo_url>
cd maximum_parser
```

### 2. Установить зависимости
```bash
pip install -r requirements.txt
```

### 3. Проверить Chrome
Нужен установленный **Google Chrome**.
**ChromeDriver**

---

## Запуск


python maxparser.py


Результат:
- изображения → `./images/`
- Excel → `maximum_data.xlsx` (имя можно поменять)
